II. Rules when to use the lower thirds.

  1. If there are no talking parts in the video, display the lowerthird all throughout the video until the end of the clip. Align the text to right.

    Example: https://www.youtube.com/watch?v=OFKuy3vj534

  2. If there are talking parts in the video, you only need to display the lowerthird the first and last 10seconds for both at the beginning and at the end of the video.

    Example: https://www.youtube.com/watch?v=t8S0piDq7m8
III. On screen title format arrangement.
Please follow the title arrangement for your project in the main title for the video. Titles should be spelled out. No Abbreviations, No Special Characters and punctuation marks.

PRIMARY TITLE
Secondary Title
Location, Season, Year

Example #1. There’s no need to add Milan on the 3rd line because it was already part in the title from the secondary line. The last line signifies the collection and year for the main subject.

ROBERTO CAVALLI
Milan Fashion Week
Autumn/Winter 2014-15

Example #2. The main subject is H&M which is the name of the Store. The name for the event is “Store Opening” which describes the primary title. The last lines states the location and year when this was happened.

H&M
Store Opening
Paris 2014

Example #3. For very long titles in the primary subejct, a two line bold and all caps are applicable.

WHALE SHARK SERIES
BY KRISTIAN SCMIDT
Art Basel Week 2014 Miami

Example #4. For very long titles at the secondary subject, a three liner is accepted.

NEW GENERATION
Young Designers Fashion Show
Mercedes-Benz Fashion Week
Russia Autumn/Winter 2014-15
